# Book A' Space UC

## Goals of Book A' Space UC
- To provide a platform for users to book spaces for various activities such as meetings, events, or workspaces. 
- We created it to make it easier for users to find and reserve spaces that suit their needs.
- We wanted it to be user-friendly and efficient, allowing users to quickly find available spaces and make reservations without hassle.

## How to Build and Run
1. Open the project in Visual Studio 2022. You can access it by navigating to the project directory and opening the solution file (.sln), or by going through GitHub and cloning the repository, then opening the solution file in Visual Studio.
2. Make sure the .NET MAUI workload is installed.
3. Restore NuGet packages.
4. Build the solution.
5. Run the project using either:
   - Windows Machine, or
   - Android Emulator
